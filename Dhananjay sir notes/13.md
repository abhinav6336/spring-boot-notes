# Paging

- mongo db insert 100 records into the collection named students

- Now create an rest api named as showrecords which show all the records through api

- The api will show all the 100 records at a time, but if we want to show only certain number of records then we can use paging.

- for implementing paging we can use the following code inside the service class.

```java
public Page<students> showdatabypage(Pageable pageable)
	{
		return sturepo.findAll(pageable);
	}
```

- inside controller write the below code-

```java
@GetMapping("/showdatabypage/{pageno}/{pagesize}")
	public Page<students> showdatabypage(@PathVariable int pageno,
										 @PathVariable int pagesize)
	{
		Pageable pageable=PageRequest.of(pageno, pagesize);
		return stuserv.showdatabypage(pageable);
	}
```
for calling 
```
	http://localhost:8888/showdatabypage/pageno/number of records
```

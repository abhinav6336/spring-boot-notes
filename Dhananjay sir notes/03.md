# DDL Command
- create - creates table etc
Syntax
```sql
	create table tablename(fieldname datatype(size),.......................................) 
```
- alter- modify the structure of existing table
Syntax
```sql
	alter table tablename add columnname datatype(size)
	alter table tablename Modify columnname datatype(size)
	alter table tablename drop column  columnname
```

- drop- delete the existing table.
Syntax
```sql
	drop table tablename
```

# DML - Data manipulation Language
- insert
Syntax
```
	a. insert into all field 
		insert into tablename(value-1,value-2,.......)
	b. insert into partial field
		insert into tablename(fieldname1,fieldname2,....) values (value-1,value-2)
```
- update
- delete

example
```sql
insert into emp values(101, 'Anuj Kapoor', 'Manager',120000)
insert into emp(empid,ename) values(102,'Pankaj Singh')
select * from emp - show all records
```
Note- insert more 5 records into emp table

emprec
```
	empid
	ename
	basic
	ta	(bs*10%)
	da    (bs*15%)
	hra   (bs*6)
```

```sql
create table emprec(empid numeric(5) primary key,ename varchar(60), basic numeric(10),ta numeric(10),da numeric(10), hra numeric(10))
```

- insert 5 records in empid,ename and basic fields only
```sql
insert into emprec(empid, ename, basic) values(101,'Anuj',70000)
insert into emprec(empid, ename, basic) values(102,'Alok',50000)
insert into emprec(empid, ename, basic) values(103,'Anu',60000)
insert into emprec(empid, ename, basic) values(104,'Amit',30000)
insert into emprec(empid, ename, basic) values(105,'Akash',40000)
```

- calculate ta,da and hra using update command
```sql
update emprec set ta=basic*10/100,da=basic*15/100,hra=basic*6/100
```

- show all the records
```sql
select * from emprec
```

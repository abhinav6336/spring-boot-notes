Thymeleaf

1. It is a default template engine of spring boot
2.  It is a html page with following convention-
<html xmlns:th="http://www.thymeleaf.org">
</html>
3. The dependency name is Thymeleaf
4. you must place the thymeleaf pages inside resources>>templates folder

home.html

<html xmlns:th="http://www.thymeleaf.org">
	<body>
		<h1>United College</h1>
		<hr/>
	</body>
</html>

Controller

public class AppController {

# CRUD Application using Thymeleaf & MySql-


- suppose we have a database named as st2026 and a table named as student is there.
student
```
	rollno
	name
	course
	branch
```
- Create a Spring Boot Project with the following dependencies-
```
	SpringBoot Dev Tools
```
# Spring Data JPA
# MySQL Driver
# Thymeleaf
```
	Spring Web
```
- add following configuration in the application.properties file-

server.port=8888
spring.datasource.url=jdbc:mysql://localhost:3306/st2026?useSSL=false
spring.datasource.username=root

- create the following packages-

- Controller
- Entity
- Repository
- Service

- Inside Entity Package create the following class-

```java
package com.training.Entity;
```

```java
import jakarta.persistence.Entity;
import jakarta.persistence.Id;
```

@Entity
```java
public class student {
	@Id
	int rollno;
	String name;
	String course;
	String branch;
	public student() {}
	public student(int rollno, String name, String course, String branch) {
		super();
		this.rollno = rollno;
		this.name = name;
		this.course = course;
		this.branch = branch;
	}
	public int getRollno() {
		return rollno;
	}
	public String getName() {
		return name;
	}
	public String getCourse() {
		return course;
	}
	public String getBranch() {
		return branch;
	}
	public void setRollno(int rollno) {
		this.rollno = rollno;
	}
	public void setName(String name) {
		this.name = name;
	}
	public void setCourse(String course) {
		this.course = course;
	}
	public void setBranch(String branch) {
		this.branch = branch;
	}
```

}

- Create a Service Class inside the Service Package

```java
package com.training.Service;
```

```java
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.stereotype.Service;
```

```java
import com.training.Entity.student;
import com.training.Repository.StudentRepository;
```

```java
@Service
public class StudentService {
```

```java
	@Autowired
	StudentRepository sturepo;
	public Iterable<student> showall()
	{
		return sturepo.findAll();
	}
```
}

- Create the Controller Class inside Controller Package

```java
package com.training.Controller;
```

```java
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.stereotype.Controller;
import org.springframework.ui.Model;
import org.springframework.web.bind.annotation.GetMapping;
import org.springframework.web.bind.annotation.RestController;
```

```java
import com.training.Entity.student;
import com.training.Service.StudentService;
```

```java
@Controller
public class AppController {
```

```java
	@Autowired
	StudentService stuserv;
	@GetMapping("/showrecords")
	public String showrecords(Model model)
	{
		model.addAttribute("records",stuserv.showall());
		return "showrecords";
	}
```


}

- create a thymeleaf page with name showrecords.html
```html
<html xmlns:th="http://www.thymeleaf.org">
	<body>
		<h1>Students Records</h1>
		<table width="500px" border="2">
			<tr style="background-color: black; color: white;">
				<td>Roll Number</td>
				<td>Name</td>
				<td>Course</td>
				<td>Branch</td>
			</tr>
			<tr th:each="row:${records}">
				<td th:text="${row.rollno}"></td>
				<td th:text="${row.name}"></td>
				<td th:text="${row.course}"></td>
				<td th:text="${row.branch}"></td>
			</tr>
		</table>
	</body>
</html>
```

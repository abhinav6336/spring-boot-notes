Introduction of Spring boot
1. Spring boot is a Java based framework, which is used for developing web applications, APIs and micro services.
2. We use java as a primary programming language

Creating a Spring Boot Application

There are multiple ways
1. spring initializr
2. VS Code
3. NetBeans
4. Eclipse
5. STS (Spring Tool Suite)

Installation 

1. JDK 17
2. spring-tool-suite-4-4.23.1

Steps for creating first spring boot application

1. File>>New>>Spring Starter Project
2. Define Project Name, select Type(Maven), Java Version 17
3. select dependency ( SpringWeb ), click on next and then finish

MVC (Model-View-Controller)

A Simple API

@Controller
@ResponseBody
public class AppController {
	@GetMapping("/info")
	public String info()
	{
		String msg="Hello Friends<br>";
		msg+="Welcome in Springboot";
		return msg;
	}
}

# Introduction of Spring boot
- Spring boot is a Java based framework, which is used for developing web applications, APIs and micro services.
- We use java as a primary programming language

# Creating a Spring Boot Application

There are multiple ways
- spring initializr
- VS Code
- NetBeans
- Eclipse
- STS (Spring Tool Suite)

# Installation

# 1. JDK 17
- spring-tool-suite-4-4.23.1

# Steps for creating first spring boot application

- File>>New>>Spring Starter Project
- Define Project Name, select Type(Maven), Java Version 17
- select dependency ( SpringWeb ), click on next and then finish

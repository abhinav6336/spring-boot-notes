# Thymeleaf

- It is a default template engine of spring boot
- It is a html page with following convention-
```html
<html xmlns:th="http://www.thymeleaf.org">
</html>
```
- The dependency name is Thymeleaf
- you must place the thymeleaf pages inside resources>>templates folder

home.html

```html
<html xmlns:th="http://www.thymeleaf.org">
	<body>
		<h1>United College</h1>
		<hr/>
	</body>
</html>
```

Controller
```java
@Controller
public class AppController {
```

```java
	@GetMapping("/home")
	public String home()
	{
		return "home";
	}
```
}

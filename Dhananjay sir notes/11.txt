MongoDB Commands-
1. show dbs - show all the existing databases
2. use databasename - if database is already created then open the database , otherwise create a new database
3. db.createCollection()- create a collection
	Syntax
		db.createCollection("collectionname")
	example
		db.createCollection("students")
4. show tables: show all the tables(collections) of the database 
	Syntax
		show tables
5. For Inserting record into collection there are two commands-
	a. insertOne() - Insert a single row
	b. insertMany() - insert multiple rows
	a. insertOne()
	Syntax
		db.collectioname.insertOne({fieldname:value,................});
			i want to store (rollno,name,course,branch)
		db.students.insertOne({rollno:101,name:'Dhananjay',course:'Phd',branch:'CS'})

	b. insertMany()
	Syntax
		db.collectionname.insertMany([{json1},{json2}])
		db.students.insertMany(	[{rollno:102,name:'Stuti',course:'B.Tech.',branch:'AIML'},{rollno:103,name:'Manavi',course:'M.Tech.',branch:'AIML'},{rollno:104,name:'Sandhya',course:'B.Tech.',branch:'CS'},{rollno:105,name:'Hema',course:'B.Tech.',branch:'IT'},{rollno:106,name:'Komal',course:'MCA.',branch:'CS'}])

6. for showing all records
		db.collectioname.find()- show all records
		example
		db.students.find()

MongoDB Operators
		Comparison Operators
			i. $eq - equals
			Syntax
				db.collectionname.find({fieldname: {operator:value}})	
				example
				db.students.find({branch: {$eq: 'CS'}}) - show all records whose branch is CS
			ii. $gt - greater than
			iii. $gte - greater than or equals to 
			iv. $lt - less than
			v. $lte - less than or equals to
			vi. $ne - not equals to 
				Questions
					Que- Show all records except the AIML branch students
							db.students.find({branch:{$ne:'AIML'}})
					Que- show all records whose rollnumber is >=103
							db.students.find({rollno:{$gt:103}})
		Logical Operators
			i. $and- Logical AND
				db.collectionname.find({$and:[{condition1},{condition-2}....]})
				Que- Show the records of all students whose course B.Tech. and branch is CS
						db.students.find({$and:[{course:{$eq:'B.Tech.'}},{branch:{$eq:'CS'}}]})
			ii. $or - Logical OR
			iii. $not - Logical Not			

		Element Operators
			$exists: check whether the particular field exist or not, if exists then show records
			example
					db.students.find({fees:{$exists:true}})

			db.students.insertOne({rollno:111,name:'Ajay',course:'B.Tech.',branch:'CS',fees:120000})
			
		Array Operator
			i. $in: check whether the field contains any of one records
			ii. $all : check whether the field contains all records
			iii. $nin
			example
				show the records of students whose course is either BCA or MCA 
				Syntax
					db.collectionname.find({fieldname:{$in:[value-1,value-2]}})

7. Record Deletion: For deleting records there also two commands-
			a. deleteOne()- delete single records
			b. deleteMany()-delete multiple records

			a. deleteOne()-
				Syntax
					db.collectionname.deleteOne(criteria)
						Example: delete the record of students whose name is Komal
								db.students.deleteOne({name:{$eq:'Komal'}})

			b. deleteMany()
				Syntax
					db.collectionname.deleteMany(criteria);			
						Example: Delete all records whose course is B.Tech.
					db.students.deleteMany({course:{$eq:'B.Tech'}}		

8. Record Updation: Updation of existing record
			a. updateOne()- update single record
			b. updateMany()- update multiple records
			
			a. updateOne()-
				Example:
				Syntax
					db.collectionname.updateOne({condition},{$set:{value}})
					Example: change the name of person as "Anuj Kapoor" whose roll number is 101.
					db.students.updateOne({rollno:101},{$set:{name:'Anuj Kapoor'}})	
9. db.collctioname.drop()- delete the particular collection
10. db.dropDatabase()- delete the particular database

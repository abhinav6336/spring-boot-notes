# Spring Boot Database and API Notes

## Request Parameters

Request parameters are used for reading values through an API using the `@RequestParam` annotation.

### Example

```java
@GetMapping("/sayhello")
public String sayhello(@RequestParam String name, @RequestParam String course) {
	String msg = "Hello " + name + "<br>";
	msg += "Your Course is " + course;
	return msg;
}
```

### Calling Syntax

```
http://localhost:8888/apiname?var1=value&var2=value&...
```

### Example URL

```
http://localhost:8888/sayhello?name=Dhananjay&course=PhD
```

## SQL Commands

### Categories

1. **DDL** (Data Definition Language)
2. **DML** (Data Manipulation Language)
3. **TCL** (Transaction Control Language)
4. **DCL** (Data Control Language)
5. **DQL** (Data Query Language)

---

## DDL Commands

### 1. CREATE - Creates tables

**Syntax:**
```sql
CREATE TABLE tablename(fieldname datatype(size), ...);
```

### 2. ALTER - Modify the structure of existing table

**Syntax:**
```sql
ALTER TABLE tablename ADD columnname datatype(size);
ALTER TABLE tablename MODIFY columnname datatype(size);
ALTER TABLE tablename DROP COLUMN columnname;
```

### 3. DROP - Delete the existing table

**Syntax:**
```sql
DROP TABLE tablename;
```

## DML - Data Manipulation Language

### 1. INSERT - Add new records

**Syntax:**

**Insert into all fields:**
```sql
INSERT INTO tablename VALUES(value-1, value-2, ...);
```

**Insert into specific fields:**
```sql
INSERT INTO tablename(fieldname1, fieldname2, ...) VALUES(value-1, value-2, ...);
```

**Examples:**
```sql
INSERT INTO emp VALUES(101, 'Anuj Kapoor', 'Manager', 120000);
INSERT INTO emp(empid, ename) VALUES(102, 'Pankaj Singh');
```

**View all records:**
```sql
SELECT * FROM emp;
```

> **Note:** Insert more 5 records into the emp table

### 2. UPDATE - Update existing records

**Syntax:**
```sql
UPDATE tablename SET columnname=value [WHERE condition];
```

### 3. DELETE - Delete existing records

**Syntax:**
```sql
DELETE FROM tablename [WHERE condition];
```

## Practice: Employee Records (EMPREC)

### Table Structure

| Field | Type | Description |
|-------|------|-------------|
| empid | numeric(5) | Employee ID (Primary Key) |
| ename | varchar(60) | Employee Name |
| basic | numeric(10) | Basic Salary |
| ta | numeric(10) | Travel Allowance (10% of basic) |
| da | numeric(10) | Dearness Allowance (15% of basic) |
| hra | numeric(10) | House Rent Allowance (6% of basic) |

### Create Table

```sql
CREATE TABLE emprec(
  empid numeric(5) PRIMARY KEY,
  ename varchar(60),
  basic numeric(10),
  ta numeric(10),
  da numeric(10),
  hra numeric(10)
);
```

### Step 1: Insert 5 records

Insert records with empid, ename, and basic fields only:

```sql
INSERT INTO emprec(empid, ename, basic) VALUES(101, 'Anuj', 70000);
INSERT INTO emprec(empid, ename, basic) VALUES(102, 'Alok', 50000);
INSERT INTO emprec(empid, ename, basic) VALUES(103, 'Anu', 60000);
INSERT INTO emprec(empid, ename, basic) VALUES(104, 'Amit', 30000);
INSERT INTO emprec(empid, ename, basic) VALUES(105, 'Akash', 40000);
```

### Step 2: Calculate Allowances

Use UPDATE command to calculate ta, da, and hra:

```sql
UPDATE emprec SET 
  ta = basic * 10 / 100,
  da = basic * 15 / 100,
  hra = basic * 6 / 100;
```

### Step 3: View All Records

```sql
SELECT * FROM emprec;
```

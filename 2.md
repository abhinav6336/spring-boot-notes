# Spring Boot API Notes

## Return Class Components Through API

### Overview
- Treat a class as a component and define it inside a Component Package
- Create a package named `component`
- Example: Create a class for processing student data

### Step 1: Create Component Class

```java
package com.training.Component;

import org.springframework.stereotype.Component;

@Component
public class student {
	int rollno, fees;
	String name, course;
	
	public student() {}
	
	public student(int rollno, int fees, String name, String course) {
		super();
		this.rollno = rollno;
		this.fees = fees;
		this.name = name;
		this.course = course;
	}
	
	public int getRollno() {
		return rollno;
	}
	
	public int getFees() {
		return fees;
	}
	
	public String getName() {
		return name;
	}
	
	public String getCourse() {
		return course;
	}
	
	public void setRollno(int rollno) {
		this.rollno = rollno;
	}
	
	public void setFees(int fees) {
		this.fees = fees;
	}
	
	public void setName(String name) {
		this.name = name;
	}
	
	public void setCourse(String course) {
		this.course = course;
	}
}
```

### Step 2: Create API Endpoint

```java
@GetMapping("/stuinfo")
public student stuinfo() {
	return new student(101, 120000, "Anuj Kapoor", "B.Tech.");
}
```

### Step 3: Call the API

```
http://localhost:8888/stuinfo
```

## Return Records of Multiple Students

```java
@GetMapping("multirec")
public List<student> multirec() {
	List<student> records = new ArrayList<>();
	records.add(new student(101, 10000, "Manoj Kumar", "B.Tech."));
	records.add(new student(102, 20000, "Ashok Kumar", "M.Tech."));
	records.add(new student(103, 30000, "Vijay Kumar", "M.Tech."));
	records.add(new student(104, 10000, "Alok Kumar", "B.Tech."));
	records.add(new student(105, 120000, "Raj Kumar", "M.Tech."));
	return records;
}
```

## Passing Values Through URL

There are two methods for passing values through URL:
1. **PathVariable**
2. **QueryString**

### Path Variables

Used for passing values directly in the URL path.

#### Syntax

```java
@GetMapping("/api/{var1}/{var2}/...")
public void apiname(@PathVariable datatype var1, @PathVariable datatype var2, ...) {
	// body of the function
}
```

#### Example

Create an API named `whomi()` that accepts name and address parameters and displays them:

```java
@GetMapping("/whomi/{name}/{address}")
public String whomi(@PathVariable String name, @PathVariable String address) {
	String msg = "Name=" + name + "<br>";
	msg += "Address=" + address;
	return msg;
}
```

#### Usage

```
http://localhost:8888/whomi/Dhananjay/Naini
```
